# Data Structure Experiment
华中科技大学网安学院大一下必修课《数据结构实验》

## 理论课编程作业1-1：链表节点的删除和添加操作
实现链表的基本操作，在链表尾部插入新节点和删除指定值的节点。

**实验内容**
1. 完成函数`insertTail`，实现在链表尾部插入一个新节点，需要满足以下要求：
* 如果链表为空（h为NULL），新节点t将成为链表的头节点
* 如果链表非空，将新节点t插入到链表的最后一个节点之后
* 返回插入操作后的链表头节点指针
* 新节点t的next指针已初始化为NULL
2. 完成函数`delNode`，实现删除链表中第一个值为e的节点，需要满足以下要求：
* 如果链表为空，不进行任何操作
* 如果头节点的值等于e，删除头节点，并将头指针指向下一个节点
* 如果链表中间某个节点的值等于e，删除该节点，并保持链表连续性
* 如果链表中没有值为e的节点，输出"error"（不含引号，换行）
* 及时释放被删除节点的内存

链表节点结构已在`linearList.h`中定义：
```
struct node {
    int data;       // 节点数据
    node* next;     // 指向下一个节点的指针
};
```

## 理论课编程作业1-2：链栈的应用——括号匹配
使用链栈实现括号匹配检查功能。给定一个包含括号字符的字符串，判断其中的括号是否匹配。

**实验内容**

完成函数`check`，实现检查字符串中的括号是否匹配，需要满足以下要求：
* 括号类型包括：(), [], {}
* 左括号 ( 必须与右括号 ) 匹配
* 左括号 [ 必须与右括号 ] 匹配
* 左括号 { 必须与右括号 } 匹配
* 括号必须以正确的顺序闭合

初始化栈、入栈和出栈操作已在`linearList.h`中定义：
```
// 初始化栈
void initstack(sqstack &s);

// 入栈操作
void push(sqstack &s, char e);

// 出栈操作，成功返回true，失败返回false
bool pop(sqstack &s, char &e);
```
