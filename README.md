# Data Structure Experiment
华中科技大学网安学院大一下必修课《数据结构实验》

## 理论课编程作业1-1：链表节点的删除和添加操作
实现链表的基本操作，在链表尾部插入新节点和删除指定值的节点。

**实验内容**
1. 完成函数`insertTail`，实现在链表尾部插入一个新节点，需要满足以下要求：
* 如果链表为空（h为NULL），新节点t将成为链表的头节点
* 如果链表非空，将新节点t插入到链表的最后一个节点之后
* 返回插入操作后的链表头节点指针
* 新节点t的next指针已初始化为NULL
2. 完成函数`delNode`，实现删除链表中第一个值为e的节点，需要满足以下要求：
* 如果链表为空，不进行任何操作
* 如果头节点的值等于e，删除头节点，并将头指针指向下一个节点
* 如果链表中间某个节点的值等于e，删除该节点，并保持链表连续性
* 如果链表中没有值为e的节点，输出"error"（不含引号，换行）
* 及时释放被删除节点的内存

**实验内容**

链表节点结构在`linearList.h`中定义，已包含：
```
struct node {
    int data;       // 节点数据
    node* next;     // 指向下一个节点的指针
};
```
